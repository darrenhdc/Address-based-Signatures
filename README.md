- ECDSA + GC2
- Address based Schnorr
- Key-prefix Schnorr 
- Key-prefix Schnorr + GC1
- Key-prefix Schnorr + GC2
- original Schnorr
- original ECDSA
- BB/BLS signature
- BB/BLS signature + GC1
- BB/BLS signature + GC2
